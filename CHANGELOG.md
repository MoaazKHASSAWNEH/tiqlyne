# Changelog

All notable changes to Tiqlyne Motion Engine will be documented in this file.

The project follows semantic versioning once the public API is stable. Before `1.0.0`, APIs may still evolve between minor versions.

## 0.1.0 - 2026-07-08

### Added

- Initial `@tiqlyne/motion-core` package.
- Initial `@tiqlyne/motion-web` package.
- Initial `@tiqlyne/motion-pack-basic` package.
- Core engine, registry, motion definitions and timeline model.
- Timeline builder, composition compiler, validation, planning and diagnostics.
- Playback contracts and playback controllers.
- Web Animations API driver.
- Basic motions: `fade-in`, `fade-out` and `slide-in`.
- Vanilla example application.
- Docusaurus documentation website.

### Notes

- This release is intended as the first public pre-release foundation.
- This release does not include official framework adapters, a visual builder, a marketplace, a GSAP driver or an advanced motion pack.
