## Current scope

Version 0.1.0 establishes registered definitions, direct timelines, compositions, planning, validation, diagnostics, controllers, events, sampling, inspection, reduced-motion policy, and Web playback.

## Packages

The release aligns `@tiqlyne/motion-core`, `@tiqlyne/motion-web`, and `@tiqlyne/motion-pack-basic`. The pack contains exactly `fade-in`, `fade-out`, and `slide-in`.

## Known limitations

Pre-1.0 APIs may evolve. Framework adapters, a visual editor, additional production drivers, and a broad preset catalogue are outside the current release.

## Documentation and examples

The official site separates getting-started docs, progressive tutorials, practical guides, copy-paste examples, exact API reference, and contributor information so each audience has a clear path.

## After 0.1.0

Later work will be considered from real integration feedback. High-level engine and builder APIs are preferred today; lower-level planner, scheduler, conversion, and validation helpers may evolve more readily.
