## I want to create and use custom motions

Use the [end-to-end tutorial](../tutorials/custom-motion-end-to-end.md) to build, register, play, debug, and inspect a motion. The shorter [definition tutorial](../tutorials/custom-motion-definition.md) introduces the class pattern. Then use:

- [Custom motion definition guide](../guides/custom-motion-definition.md) for production patterns.
- [Registered motions guide](../guides/registered-motions.md) for registry assembly.
- [Motion definition](../reference/motion-definition.md) and [motion registry](../reference/motion-registry.md) for exact contracts.
- [Motion options](../reference/motion-options.md) and [timeline builder](../reference/timeline-builder.md) for authoring details.
