# Version 0.1.0 boundaries

Version 0.1.0 is a typed foundation, not a complete animation ecosystem.

Available now: framework-independent timelines, registered definitions, compositions, planning and validation, diagnostics, sampling and inspection, browser execution through WAAPI, controllers, and the three basic motions `fade-in`, `fade-out`, and `slide-in`.

Not included: framework adapters, a visual editor, a marketplace, a GSAP driver, non-browser production drivers, or a large preset catalogue. Browser target references are resolved only by `self`, `child`, `selector`, and `named` rules. See [Limitations](../release/limitations.md) for runtime details.

Public APIs may still evolve before 1.0. Pin exact package versions when stability is important.
