# Public exports

This page summarizes the main public exports of the official packages.

## @tiqlyne/motion-core

Main exports include:

- `createMotionEngine`
- `MotionEngine`
- `DefaultMotionEngine`
- `DefaultMotionRegistry`
- `MotionDriver`
- `MotionDefinition`
- `BaseMotionDefinition`
- `SchemaMotionDefinition`
- `createMotionTimeline`
- `createMotionComposition`
- `compileMotionComposition`
- `sampleMotionTimeline`
- `sampleMotionTimelineAt`
- `inspectMotionTimeline`
- playback controller types
- diagnostics helpers
- option builders
- validators

## @tiqlyne/motion-web

Main exports include:

- `WebMotionDriver`
- `WebMotionDriverOptions`
- `WebMotionPlaybackController`
- web keyframe helpers
- web timing helpers
- target resolution helpers
- reduced motion helpers
- conflict helpers

## @tiqlyne/motion-pack-basic

Main exports include:

- `FadeInMotion`
- `FadeOutMotion`
- `SlideInMotion`
- `registerBasicMotions`
- `motionPackBasicVersion`

## Stability note

The documentation targets version `0.1.0`.
