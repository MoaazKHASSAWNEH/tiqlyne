# Version 0.1.0

Version `0.1.0` is the first public pre-release foundation of Tiqlyne Motion Engine.

## Included packages

- `@tiqlyne/motion-core`
- `@tiqlyne/motion-web`
- `@tiqlyne/motion-pack-basic`

## Main capabilities

- engine creation
- registry
- registered motions
- direct timelines
- compositions
- planning
- validation
- diagnostics
- playback controllers
- playback events
- sampler
- inspector
- reduced motion support
- Web Animations API driver
- basic motion pack

## Basic pack motions

- `fade-in`
- `fade-out`
- `slide-in`

## Not included

This version does not include a visual builder, marketplace, official framework adapters, GSAP driver or advanced motion pack.

## Release goal

The goal is to provide a stable and documented foundation that other packages, drivers and tools can build on.
