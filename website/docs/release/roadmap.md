# Roadmap

The roadmap keeps the project focused and avoids adding too much scope too early.

## Version 0.1.0

The first public pre-release focuses on the foundation.

Goals:

- publish official packages under `@tiqlyne`
- provide official documentation
- validate package exports
- document the current API
- provide browser playback through `@tiqlyne/motion-web`
- provide basic motions through `@tiqlyne/motion-pack-basic`

## Version 0.1.x

Possible follow-up improvements:

- more examples
- more diagnostics documentation
- improved sampler documentation
- additional basic motions
- documentation refinements
- package metadata improvements

## Future versions

Future versions may explore framework adapters, advanced motion packs, development tools, visual authoring tools and additional platform drivers.

## Out of scope

The first release does not include a visual builder, marketplace, official framework adapters, GSAP driver or advanced motion pack.
