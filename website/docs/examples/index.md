# Examples

These are copy-paste examples for application developers. They use published `@tiqlyne` package APIs and show the HTML and TypeScript needed for a focused use case.

Repository demos and monorepo commands are intentionally kept in [Project: internal vanilla example](../project/internal-vanilla-example.md).

| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Basic entrances and exits  | [Fade in](./fade-in.md), [fade out](./fade-out.md), [slide in](./slide-in.md)                                                                   |
| Nested or repeated targets | [Child targets](./child-targets.md), [staggered list](./stagger-list.md)                                                                        |
| Custom sequencing          | [Direct timeline](./direct-timeline.md), [multiple tracks](./multiple-tracks.md), [labels](./labels.md)                                         |
| Runtime behavior           | [Composition](./composition.md), [controller](./playback-controller.md), [reduced motion](./reduced-motion.md), [diagnostics](./diagnostics.md) |
